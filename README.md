# Weather Forecast App

A responsive Weather Forecast Web Application built using HTML, CSS, and JavaScript. This project allows users to search for the current weather and weather forecast of any city using a weather API. The application provides real-time weather information through a clean and user-friendly interface.

---

## Features

- Search weather by city name
- View current weather conditions
- View weather forecast for multiple days
- Display minimum and maximum temperatures
- View humidity information
- Display wind speed and wind direction
- Show atmospheric pressure
- Responsive design
- Simple and user-friendly interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API

## Usage

1. Open the application.
2. Enter the desired city name.
3. Click the Search button.
4. View the current weather details.
5. Navigate to the Forecast page to view upcoming weather conditions.
